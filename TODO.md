Sidewall slot G code: 	tabs are not currently generated correctly, and toolpaths to widen slot to specified dimension are not generated. Make it so the machine origin in the gcode is in the position specified in the core blank setup tab.

Add 3D toolpath visualisation to the G-code tab, allowing the view to be rotated with the right mouse button, panned with the middle (scroll wheel) mouse button and zoomed with the mouse scroll wheel.

Add base design tab after the outline design tab and before the core geometry tab. This should take the ski planform design (or reads a ski definition file) and allows the user to set the edge width and tip and tail offsets. The metal edges do not wrap all the way round the ski, so these define the locations at which the metal edges start and finish on each side of the ski. If either value is set to zero, that means the edge does wrap around the corresponding and there is only one strip of metal in the ski rather than two distint strips, one for each side. The tab should take the ski outline and add the cutouts for the metal edges. This is quite similar to the offsets to accomodate the sidewalls added to (or rather subtracted from) the core edges. The tab should give a visualisation of the base outline, including edge cutouts, and export the base geometry to a DXF file. Allow G code to be generated to cut out the base shape either with either a CNC router or drag knife. This will need the base material size to be input (length, width, thickness - default to 2mm thick). Allow the ski definition file to be saved, with the parameters relevent to the base design in a new sub-array.

Add a core profiling tab after the sidewall slots gode tab. This should generate the gcode required to machine the thickness profile into the core/sidewall combo(s) after the sidewalls have been glued into the slots formed in the previous step. Add options for the toolpaths to run either along or across the ski. It should generate a roughing and a finishing pass. Add options for depth of cut per pass, stepover, tool diameter etc, and finishing pass depth. Add a similar toolpath visualisation and playback pane as in the existing sidewall slots g code tab.

Add a tab to define the camber design of the ski. The vertical shape of the ski is defined by three sections: tip rocker, camber underfoot and tail rocker. Add inputs for tip and tail rocker lengths and tip and tail heights and amount of camber. The ski contacts the snow at the end of the tip and tail rocker sections. Define the camber line with three splines, one for each section. Make a visualisation of the camberline in the top visualisation pane, similar to the Design tab.  Show the control points and allow them to be dragged. The control arms at the intersections of the tip and tail rocker sections should be constrained to be horizontal.
